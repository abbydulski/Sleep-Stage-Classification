Welcome to the Sleep Classification Model

This is a project developed out of the Fluid Interfaces group at MIT.

What is this project?

This model uses machine learning to classify sleep stages based on data from any wearable that can provide HR data and Accelerometer data. The goal of this site is to make sleep classification more accessible to the average person with a wearable, as opposed to in-depth analysis in a sleep lab.
